# cobyformer

This repo has an autograd engine, transformer, and some basic pytorch-y OOP setup that I've written by hand for the purposes of (1) improving at numpy and Python OOP and (2) strongly internalizing how ML frameworks work so that I can have extremely accurate mental models of my code when doing research.

I typed all of the code in here with my fingers and without the use of LLMs (that would defeat the purpose). I did use LLMs for some conceptual help, asked questions about syntax, and asked for architectural/algorithmic feedback about implementations only after getting them to work correctly.
